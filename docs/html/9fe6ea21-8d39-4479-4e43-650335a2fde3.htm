      Vec3T Structure

      Fixed-size vector3 struct with special reloaded operators and optimized loop speeds.
      Absolute value (Abs(a)) =========== +a
      Normalized value (a.Normalized()): === ~a
      Dot product (a·b) =============== a * b
      Angle between ((a·b)/(|a|*|b|)) ======== a / b
      Cross product (a X b) ============ a % b
