      IndexArrayT Operators and Type Conversions

      Operators

      Equality(IndexArrayT, IndexArrayT) Compars the equality of both operands using the overloaded Equals operator.
      (T to IndexArrayT) Explicits converts an array of T to an IndexVar<T>
      (IndexArrayT to T) Explicits converts an IndexVar<T> to an array of T
      Inequality(IndexArrayT, IndexArrayT) Compars the inequality of both operands using the overloaded Equals operator.
