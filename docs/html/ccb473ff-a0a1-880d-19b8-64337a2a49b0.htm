      RanSharp.Performance Namespace

      The Performance namespace contains custom datatypes and extension methods that are performant. FastList is a List but with faster looping speed, and with direct access to its inner array. Loop and NumericLoop provides extension methods for easy looping operations. Calc provides abilities to use the System.Maths library on generic type T. ListExtensions and ArrayExtensions provide extension methods to Lists and Arrays such as more readable string representations, and convenient looping operations such as Composite or Map.

      Classes

      ArrayExtensions A class to provide extension methods for arrays.
      CalcT A class that contains static methods for performing calculations on values of type T.
      FastListT A fast list that is not thread-safe. It provides a subset of the functionality of ListT. It is faster than ListT because it directly exposes the underlying array in the ItemsUnsafe property, allowing for fast iteration using SpanT or unsafe code.
      ListExtensions A class that provides extension methods for lists.
      Loop A class that contains static methods for looping over values.
      LoopT A class that contains static methods for looping over values of type T.
      NumericLoopT A static class containing methods for looping over arrays of numeric types T.
