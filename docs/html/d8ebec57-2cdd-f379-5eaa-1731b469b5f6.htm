      ArrVectorT Operators and Type Conversions

      Operators

      Addition(T, ArrVectorT) Adds scalar a to each element of b
      Addition(ArrVectorT, T) Adds scalar b to each element of a
      Addition(ArrVectorT, ArrVectorT) Adds a and b
      Decrement(ArrVectorT) Decrements each element of a
      Division(T, ArrVectorT) Divides each element of b by scalar a
      Division(ArrVectorT, T) Divides each element of a by scalar b
      Division(ArrVectorT, ArrVectorT) The angle between a and b
      Equality(ArrVectorT, ArrVectorT) Returns true if a and b are equal
      ExclusiveOr(T, ArrVectorT) Raises each element of b to the power of scalar a
      ExclusiveOr(ArrVectorT, T) Raises each element of a to the power of scalar b
      GreaterThan(ArrVectorT, ArrVectorT) Returns true if the magnitude of a is greater than the magnitude of b
      (T to ArrVectorT) Converts the given array to a ArrVectorT implicitly.
      (ArrVectorT to T) Converts the given ArrVectorT to an array implicitly.
      (ArrVectorT to ListT) Converts the given ArrVectorT to a ListT implicitly.
      (ArrVectorT to ReadOnlySpanT) Converts the given ArrVectorT to a ReadOnlySpanT implicitly. Note that this ReadOnlySpanT keeps the reference to the original internal array. Looping over it will have the same effect as looping over the original ArrVectorT.
      (ArrVectorT to SpanT) Converts the given ArrVectorT to a SpanT implicitly. Note that this SpanT keeps the reference to the original internal array. Modifying it will have the same effect as modifying the original ArrVectorT.
      (ListT to ArrVectorT) Converts the given ListT to a ArrVectorT implicitly.
      (ReadOnlySpanT to ArrVectorT) Converts the given ReadOnlySpanT to a ArrVectorT implicitly.
      (SpanT to ArrVectorT) Converts the given SpanT to a ArrVectorT implicitly.
      Increment(ArrVectorT) Increments each element of a
      Inequality(ArrVectorT, ArrVectorT) Returns true if a and b are not equal
      LessThan(ArrVectorT, ArrVectorT) Returns true if the magnitude of a is less than the magnitude of b
      Modulus(ArrVectorT, ArrVectorT) Cross product of a and b
      Multiply(T, ArrVectorT) Multiplies scalar a to each element of b
      Multiply(ArrVectorT, T) Multiplies scalar b to each element of a
      Multiply(ArrVectorT, ArrVectorT) Dot product of a and b
      OnesComplement(ArrVectorT) Normalized value of a
      Subtraction(T, ArrVectorT) Subtracts scalar a from each element of b
      Subtraction(ArrVectorT, T) Subtracts scalar b from each element of a
      Subtraction(ArrVectorT, ArrVectorT) Subtracts b from a
      UnaryNegation(ArrVectorT) Negated value of a
      UnaryPlus(ArrVectorT) Absolute value of a
