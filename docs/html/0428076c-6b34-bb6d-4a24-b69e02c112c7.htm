      LineT(T, T) Creates a new line between 2 points. It has no attributes attached. Thus, accessing the indexer will throw an exception.
      LineT(T, T, Int32, Int32) Creates a new line between 2 points. It is also possible to add attributes to the line. It is advised to implement an enum (specifying their Int32 values as well) for the attributes.
