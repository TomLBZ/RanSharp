      Mat3D   Conversion Operators

      Overload List

      (D to Mat3D) Returns a new matrix with the given array of values.
      (D to Mat3D) Returns a new matrix with the given 2d array of values.
      (Mat3D to D) Returns an array of the values in the matrix.
      (Mat3D to D) Returns a 2d array of the values in the matrix.
      (Mat3D to Vec3D) Returns an array of the rows in the matrix.
      (Vec3D to Mat3D) Returns a new matrix with the given array of rows.
