      FastListT Properties

      Properties

      Capacity Gets or sets the capacity of the FastListT.
      Count Gets the number of elements contained in the FastListT.
      IsReadOnly Always returns false because FastListT is not read-only.
      Item The indexer of the FastListT. This allows direct access to available indexes in the FastListT before the Add(T) has been called, as long as the length of the inner array has not been exhausted. However, you should not depend on this behaviour unless you know what you are doing.
      ItemsUnsafe Directly exposes the inner array. This allows direct access to the inner array, but it is not recommended to use this property unless you know what you are doing. The FastListT will not be able to track changes made to the inner array.
