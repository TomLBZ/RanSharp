      FastListT Constructor

      Overload List

      FastListT Initializes a new instance of the FastListT class that is empty and has the default initial capacity.
      FastListT(T) Initializes a new instance of the FastListT class that contains elements copied from the specified T[] and has sufficient capacity to accommodate the number of elements copied.
      FastListT(IEnumerableT) Initializes a new instance of the FastListT class that contains elements copied from the specified IEnumerableT and has sufficient capacity to accommodate the number of elements copied.
      FastListT(Int32) Initializes a new instance of the FastListT class that is empty and has the specified initial capacity.
      FastListT(Int32, FuncInt32, T) Initializes a new instance of the FastListT class that has the specified initial capacity and each element is generated by the specified generator.
